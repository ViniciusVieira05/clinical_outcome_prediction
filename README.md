# Clinical Outcome Prediction using Machine Learning

Projeto de Inteligência Artificial aplicado à área da saúde para previsão de desfechos clínicos utilizando Regressão Logística e Random Forest.

---

# Disciplina

Este trabalho foi desenvolvido para a disciplina de:

## Inteligência Artificial — Módulo III  
Curso: **Análise e Desenvolvimento de Sistemas (ADS)**

---

# Sobre o Projeto

Este projeto tem como objetivo prever desfechos clínicos de pacientes com base em dados hospitalares, utilizando técnicas de pré-processamento, balanceamento de dados e modelos de classificação supervisionada.

Os desfechos clínicos previstos foram:

- Stable
- Readmitted
- Complication

O projeto também realiza comparação entre modelos de Machine Learning considerando desempenho e interpretabilidade.

---

# Objetivos

- Aplicar técnicas de pré-processamento em dados clínicos
- Realizar tratamento de variáveis categóricas
- Balancear classes desbalanceadas com SMOTE
- Treinar modelos de classificação
- Comparar Regressão Logística e Random Forest
- Avaliar modelos com métricas clínicas e estatísticas

---

# Modelos Utilizados

## Regressão Logística
Modelo linear interpretável utilizado para classificação dos desfechos clínicos.

## Random Forest
Modelo ensemble baseado em múltiplas árvores de decisão para melhorar capacidade preditiva.
